# car-price-prediction
A Machine Learning-based Car Price Prediction system that predicts the selling price of used cars based on features like company, model, year, fuel type, and kilometers driven.
This project is a Machine Learning web application that predicts the price of a used car based on various input features such as:
Car Company
Car Model
Manufacturing Year
Fuel Type
Kilometers Driven
The project uses data preprocessing, cleaning, and machine learning algorithms to provide accurate car price predictions. The frontend is built with HTML/CSS, and the backend is powered by Python Flask.
Features
✔ Predicts car prices instantly
✔ User-friendly web interface
✔ Data cleaning and preprocessing
✔ Machine Learning model integration
✔ Real-time prediction results
Tech Stack
Python
Flask
HTML
CSS
JavaScript
Pandas
Scikit-learn
Project Goal
The goal of this project is to help users estimate the market value of used cars quickly and accurately using Machine Learning.
